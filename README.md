# 🎵 StreamAudioCore

**StreamAudioCore** هو مشروع برمجي متكامل يعتمد على تقنية **Tauri** لإنشاء تطبيقات سطح مكتب سريعة وفعالة ومبهرة بصرياً. يوفر المشروع بيئة متقدمة لمعالجة وتشغيل الصوتيات والوسائط بأداء عالٍ جداً بفضل النواة المبرمجة بلغة **Rust**.

---

## ✨ الميزات الرئيسية

* 🚀 **أداء فائق:** مبني على محرك **Tauri** (Rust) مما يضمن استهلاكاً منخفضاً للذاكرة وسرعة استجابة مذهلة مقارنة بالتقنيات الأخرى.
* 🎨 **واجهة عصرية:** واجهة مستخدم (UI) احترافية مبنية باستخدام **React**, **TypeScript**, و **Vite** لضمان أفضل تجربة مستخدم (UX).
* 🌊 **عرض الموجات الصوتية (Waveform):** معالجة وعرض الموجات الصوتية بشكل دقيق وسلس.
* 🎛️ **تحكم متقدم بالصوت:** إمكانية التحكم المتقدم في قنوات الصوت واختيار وتخصيص أجهزة الإخراج (Output Devices) بسلاسة.
* 🔠 **خطوط مخصصة مدمجة:** تم تضمين خطوط احترافية مدفوعة لتعزيز جمالية العرض (موجودة في مجلد `front/`).

## 🛠️ إعداد بيئة التطوير والتشغيل

لتتمكن من تشغيل أو تطوير المشروع على جهازك، تأكد من توفر المتطلبات الأساسية التالية:

1. **[Node.js](https://nodejs.org/):** (الإصدار 18 أو أحدث).
2. **[Rust & Cargo](https://rustup.rs/):** المعالج الخاص بنواة التطبيق الخلفية.
3. **أدوات البناء (C++ Build Tools):** لنظام ويندوز (Visual Studio Build Tools مع دعم C++).

### 🚀 خطوات التشغيل

1. قم بالدخول إلى مجلد التطبيق الأساسي عبر سطر الأوامر:
   ```bash
   cd streamaudio-core
   ```
2. قم بتثبيت حزم النود والاعتماديات:
   ```bash
   npm install
   ```
3. قم بتشغيل بيئة التطوير (هذا الأمر سيقوم تلقائياً بتشغيل خادم Vite وبناء تطبيق Tauri):
   ```bash
   npm run tauri dev
   ```
